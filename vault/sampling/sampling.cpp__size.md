# sampling.cpp__size

### size() function

Returns the size of the object.

This function is a simple getter that returns the size of the object, which is stored in the `sz` variable. It is a const function, meaning it does not modify the object's state.

#### Performance

This function has a constant time complexity of O(1), as it simply returns a stored value.
